# Create a single app using Create React App CLI

## Using create-react-app CLI to create a new project with Typescript Template

<pre>
npx create-react-app my-app --template typescript
</pre>

## Integrate CSS framework (Bootstrap, Semantic UI, Ant …)
Sử dụng Antd
<pre>
npm i antd
</pre>

## Using SCSS instead of CSS
Trước tiên cần phải cài đặt sass trước
<pre>
npm i sass
</pre>
